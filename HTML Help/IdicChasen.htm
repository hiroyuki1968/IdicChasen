アイドル相性ランキング Ver.1.23 ヘルプ

E-Mail:   hashikun777@hotmail.com
HP: http://hp.vector.co.jp/authors/VA030717/



概要

　「アイドル相性ランキング」は、あなたと、オンラインソフトの「I-dic（アイドル人名辞書）」の全アイドルとの、相性（恋愛関係、友人関係、家族関係、金銭関係、師弟関係、ビジネス関係、ライバル関係）を、オンラインソフトの「チャート占星術」を使って求めて、ランキング表示するソフトウェアです。

　また、「チャート占星術」の日運を、毎回、姓名、性別、生年月日、出生時、出生地、世界時との時差、を入力しなくても、本ソフトウェアによる自動入力によって表示させることができます。

　プログラムは VC++ で作成されています。

動作環境

    OS   Windows 95、Windows 98、Windows Me、Windows NT、Windows 2000、Windows XP、Windows 2003
 
    グラフィック   16 ビットカラー（65,536 色）以上
 
ブラウザ Internet Explorer 5.0 以降
  • 「アイドル相性ランキング」の「最新の Yahoo! 芸能、タレント名のダウンロード」の実行には Internet Explorer がインストール＆接続設定されている必要があります。

インストール方法

　このファイルのあるフォルダ内のファイル setup.exe を実行し、表示されるウィザードの指示に従ってインストールを行います。

　「Windows Installer」のアップグレードが必要な場合、インストーラーが自動でアップグレードパッケージのダウンロードとインストールを行います。

アンインストール方法

　スタートメニューの

　　　「プログラム」→「アイドル相性ランキング」→「アイドル相性ランキング のアンインストール」

または、コントロールパネルの

　　　「アプリケーションの追加と削除」

によりアンインストールできます。

アップグレード方法

　新しいバージョンのインストールを実行します。古いバージョンのアンインストールは自動で行われます。

「チャート占星術」の準備

　Vector のホームページの http://www.vector.co.jp/vpack/browse/person/an013699.html の「チャート占星術」をダウンロードし、展開します。

「I-dic（アイドル人名辞書）」の準備

　Vector のホームページの http://www.vector.co.jp/vpack/browse/person/an000877.html の「I-dic」の「I-dic idc???????.lzh」（FEP 版でなく、かつ、lzh 圧縮版）をダウンロードし、展開します。

起動方法

　スタートメニューの

　　　「プログラム」→「アイドル相性ランキング」→「アイドル相性ランキング」

または、デスクトップ上の

　　　「アイドル相性ランキング」アイコン

をクリックします。

「チャート占星術」の実行ファイルchasen.exeのファイルパスの設定



　「チャート占星術」の実行ファイル chasen.exe のファイルパスを入力してください。

「I-dic（アイドル人名辞書）」の辞書ファイルi-dicのファイルパスの設定



　「I-dic（アイドル人名辞書）」の辞書ファイル i-dic（拡張子なし）のファイルパスを入力してください。

姓名、性別、生年月日、出生時、出生地、世界時との時差、の入力



　姓名、性別、生年月日、出生時、出生地、世界時との時差、を入力してください。

実行



　「日運」チェックボックスをオフにし、をクリックしてください。



　実行中に「チャート占星術」の終了をしたり、相性値グラフが画面に表示されないようにすると、「アイドル相性ランキング」がエラーを起こす場合があります。



　実行中は「チャート占星術」ウィンドウは常に手前に表示されるようになります。



　実行結果はリストコントロールに

　　合計 ＞ 恋愛 ＞ 友人 ＞ 家族 ＞ 金銭 ＞ 師弟 ＞ ビジネス ＞ ライバル ＞ 姓名

でソートされて表示されます。

ソート



　実行結果が表示されたリストコントロールの列のタイトル（姓名、生年月日 年齢、血液型、恋愛、友人、家族、金銭、師弟、ビジネス、ライバル、合計）をクリックすると、その列をキーにソートできます。同じ列のタイトルを再度クリックすると逆順ソートできます。

実行結果の表中のアイドルの運勢の表示



　実行結果が表示されたリストコントロールのアイドル名をダブルクリックすると、「チャート占星術」ウィンドウにそのアイドルの運勢が表示されます。

Yahoo!の「芸能、タレント」に登録されているアイドルのみの表示





　をクリックすると、「Yahoo!」の「芸能、タレント」に登録されているアイドル名をダウンロードします。ダウンロードが完了すると、「Yahoo! 登録アイドル」チェックボックスがオン／オフできるようになります。

　「Yahoo! 登録アイドル」チェックボックスをオンにすると、リストコントロールに表示されるアイドルは、「Yahoo!」の「芸能、タレント」に登録されているアイドルのみになります。

　「Yahoo!」の「芸能、タレント」からダウンロードされるカテゴリは、
  • 芸能、タレント
  • 芸能、タレント/アナウンサー
  • 芸能、タレント/グラビアアイドル
  • 芸能、タレント/俳優、女優
  • 芸能、タレント/モデル
  • 芸能、タレント/アーティスト/ジャパニーズポップス
です。

異性のみの表示



　「アイドルの性別」の「異性」チェックボックスをオンにすると、リストコントロールに表示されるアイドルは異性のみになります。

表示するアイドルの年齢の上限、下限指定



　「アイドルの年齢」のテキストボックスで、表示するアイドルの年齢の上限、下限を指定できます。[ENTER] キーの入力か、フォーカスの移動で、リストコントロールに反映されます。

表示するアイドルの血液型指定



　「アイドルの血液型」のチェックボックスで、表示するアイドルの血液型を指定できます。

検索

　 ボタンをクリックすると、検索結果から姓名で検索をすることができます。


CSV、INIファイル書き出し

　 ボタンをクリックすると、実行結果を CSV ファイルに、コントロール入力値を INI ファイルに保存できます。

CSV、INIファイル読み込み

　 ボタンをクリックすると、保存した実行結果の CSV ファイルと、コントロール入力値の INI ファイルを読み込むことができます。（INI ファイルが存在しない場合、実行結果の CSV ファイルのみ読み込みます。）

日運



　「日運」チェックボックスをオンにし、をクリックすると、「チャート占星術」ウィンドウに日運が表示されます。

トラブルシューティング

　「チャート占星術」のバージョンアップにより、実行結果が正しく表示されなくなる場合があります。その際は「アイドル相性ランキング」の最新版をダウンロード、インストールしてください。

最新版の入手先ホームページ
